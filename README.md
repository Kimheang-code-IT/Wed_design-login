
2. Open `HTML/index.html` in your web browser

3. Start using the application!

## Usage

### Login
- Enter your registered email and password
- Click "Login" to access the user management system

### Sign Up
- Click "Sign up" on the login page
- Fill in your name, email, and password
- Confirm your password
- Click "Create Account"

### Forgot Password
- Click "Forgot password?" on the login page
- Enter your registered email
- Your password will be displayed

### User Management
- View all users in the table
- Click "Add New User" to create a new user
- Click "Update" to edit a user
- Click "Delete" to remove a user
- Click "Logout" to return to login page
